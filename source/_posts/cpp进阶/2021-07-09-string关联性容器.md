### 字符串

#### C风格

`char st[100];`

* `size_t strlen(const char *str)`字符串长度

* `int strcmp( const char *lhs, const char *rhs )`, `int strncmp( const char *lhs, const char *rhs, size_t count );` 把st1,st2的前count个进行比较

* `char *strcat( char *dest, const char *src )`, Appends a copy of the null-terminated byte string pointed to by src to the end of the null-terminated byte string pointed to by dest, returns a copy of dest。`char *strncat( char *dest, const char *src, size_t count );`

* `char *strcpy(char *dest, const char *src)` 复制或者覆盖。`char * strncpy ( char * destination, const char * source, size_t num )`。

* `const char * strchr ( const char * str, int character );` 查找字符，返回找到第一个字符的指针

* `const char * strstr ( const char * str1, const char * str2 );`在str1中查找匹配字符串str2

```cpp
#include <stdio.h>
#include <string.h>

int main ()
{
  char str[] ="This is a simple string";
  char * pch;
  pch = strstr (str,"simple");
  if (pch != NULL)
    strncpy (pch,"sample",6);
  puts (str);
  return 0;
}

// 输出
This is a sample string
```

### 仿函数

#### 仿函数

所谓的仿函数，就是基于`operator()`写法的class closure。

例如算术类仿函数
```cpp
template <class T>
struct plus : public binary_function<T> {
    T operator() (const T& x, const T& y) const {
        return x + y;
    }
};

template <class T>
struct modulus : public binary_function<T> {
    T operator() (const T& x, const T& y) const {
        return x % y;
    }
};

// 使用
plus<int> plusobj;
cout << plusobj(3,5) << endl;

accumulate(iv,begin(), iv.end(), plus<int>());

```

关系类仿函数

* 等于 equal_to<T>
* 大于 greater<T>
* 小于 less<T>
* 不等于 not_equal_to<T>
* 大于等于 greater_equal<T>
* 小于等于 less_equal<T>

```cpp
template <class T>
struct equal_to : public binary_function<T> {
    T operator() (const T& x, const T& y) const {
        return x == y;
    }
};
```

#### 配接器

所谓的配接器，功能类似于`std::bind`,而仿函数功能类似于`std::function<>`。配接器和仿函数的共同作为类似于`std::bind`与`std::function<>`的共同作用。这将一个函数或者成员函数表现成表达式的形式。

`not1(bind2nd(less<int>(), 12))`该表达式首先绑定`less<int>()`的参数为12，表示小于12。然后`not1`表示取反，意思是不小于12。

`count_if(iv.begin(), iv.end(), not1(bind2nd(less<int>(), 12)))`表示不小于12的元素个数。

```cpp
#include <iostream>
#include <vector>
#include <alogrithm>
#include <functional>
using namespace std;

class Shape
{ public: virtual void display() = 0; };

class Rect : public Shape
{ 
public: 
    virtual void display(){
        cout << "Rect;"<<endl;
    }
};

class Circle : public Shape
{ 
public: 
    virtual void display(){
        cout << "Circle;"<<endl;
    }
};

int main() {
    vector<Shape*> V;
    v.push_back(new Rect);
    v.push_back(new Circle);

    for_each(V.begin(), V.end(), mem_fun(&shape::display));
    return 0;
}
```
