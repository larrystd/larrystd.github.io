> 记录经典的笔试真题

#### 4.17网易互娱 T2

大体描述是, 阴阳师有六个装备槽, 每个装备槽有几个备选装备(类似槽A备选装备A1, A2.., 槽B备选装备B1, B2... 等)每个装备简单认为只有两种特征: 暴击率和攻击力。问使六个装备暴击率之和大于100的最大攻击力, 如果没有暴击率之和>100的选择返回-1.

该题暴力回溯只能通过70%

这个题形式上类似背包问题, 关键在于怎样求在暴击率大于100的条件下攻击力最大。

令dp[i][k]表示选择了0~i的装备槽且暴击率为k条件下的攻击力最大值, 那么我们只需要遍历当k>=100下dp[6][k]值就可以得到结果
```
for(int k=100;k<=150;k++) 
  ans = max(ans, dp[6][k]);
```

动态规划代码
```cpp
for(int i=1;i<=6;i++) // 6个装备槽
{
    memset(f, -1, sizeof(f));
      
    for(int j=1;j<=len[i];j++)  // 第i个槽备选的装备数为len[i]
    {
        for(int k=150;k>=op[i][j].c;k--)
        {
            if(dp[i-1][k-op[i][j].c] >= 0)
            f[j][k] = max(f[j][k], dp[i-1][k-op[i][j].c] + op[i][j].d); // f[j][k]表示截至到第i个装备槽选第j个, 攻击力的最大值。因此需要dp[i-1][k-op[i][j].c] + op[i][j].d 
        }
    }
    for(int k=0;k<=150;k++) dp[i][k] = dp[i-1][k];
    for(int k=0;k<=150;k++)
    {
        for(int j=1;j<=len[i];j++)  // f[j][k] 对j方向的最大值设置为dp[i][k]
          dp[i][k] = max(dp[i][k], f[j][k]);
    }
    int ans = -1;
    for(int k=100;k<=150;k++) ans = max(ans, dp[6][k]);
    if(ans == -1) cout<<"TAT"<<endl;
    else cout<<ans<<endl;
}
```
