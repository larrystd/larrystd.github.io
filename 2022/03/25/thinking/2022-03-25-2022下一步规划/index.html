碎碎念
思考

3.25

很幸运拿到了阿里云存储部门的暑期实习意向书, 接下来针对秋招和工作相关的, 目前想着计划如下

计算机课程

更加深入地, 从实践中理解计算机, 数据库, 分布式等。采取的方法是快速做或参考一些国外优质计算机课程和完成课后lab。这些课程十分有利于从实践中理解相关知识, 而不再仅限于书中。包括但不限于

  1. mit 6.824 Distributed System, go语言。 包括四个lab, 分别对应于mapreduce, raft, raft kv store, 负载均衡和分片raft kv store

并发程序的调试是一个难点，基本上只能通过打日志的方式找到问题，需要耗费比较多的时间，因此实现的过程中尽量多输出日志。lab2 需要对 Raft 论文理解完全到位，否则会踩坑。lab4 需要自己设计迁移数据的方法。lab2、lab3、lab4 之间存在关联，后一个 lab 依赖于前一个 lab 的正确性，这三个 lab 合起来是一个完整的分布式键值存储系统。

  1. cmu 15-213/14-513/15-513 Intro to Computer Systems。lab主要有Bomb Lab(汇编), Attack Lab, Cache lab, Malloc lab等

  2. CMU 15-445, 数据库课程, Project1：实现一个BufferPoolManager, project2：实现B+树, peoject3：实现Query Exeution的支持,火山模型, project4：支持并发的查询执行

  3. MIT6.S081 Operating System Engineering, 操作系统进阶课程, Lab2 System calls, Lab3 Page tables, Lab4 Traps,
    Lab6 Copy-on-write fork, Lab9 File System

开源项目

主要是数据库项目

  1. 关系型数据库, MySQL, PostgreSQL

  2. 列式存储, 分析型数据库clickHouse (mysql是行式存储)

  3. 分布式事务数据库TiDB, 存储引擎TiKV; 以及oceanbase polardb

  4. 云原生kubernets

  5. 分布式存储HDFS, Ceph

4.11

一些招聘的要求

蚂蚁集团-OceanBase存储引擎研发专家-杭州/上海 3年经验

1、计算机相关专业本科及以上学历，熟悉C/C++/java/python/go任意语言
2、有大规模分布式存储/缓存/数据库系统开发设计经验，了解相关开源项目架构，熟悉Oracle/MySQL/Mariadb/PostgreSQL等优先
3、熟悉hbase/spark/cockroachdb/tidb/kylin等开源分布式数据库项目的优先。
4、熟悉redis/rocksdb/memcached/leveldb等开源kv数据库项目的优先。
5、研读过lsm tree/f1/spanner/aurora/paxos/raft等相关论文的优先。
6、良好的工程质量意识，熟悉单元测试、功能测试和系统测试；
7、追求卓越，自我驱动，对代码工程质量有近乎洁癖的自我要求。
8、良好的沟通能力与团队合作意识。

蚂蚁集团-OceanBase分布式文件系统工程师/专家-杭州 3年经验

  1. 熟练使用C/C++，有多线程、异步编程经验，有扎实的算法基础，熟悉常见的数据结构；
  2. 熟悉文件系统、存储系统、分布式算法、计算机网络，理解大型分布式存储系统工作原理；
  3. 有文件存储、块存储、对象存储开发和使用经验(Ceph、HDFS、Lustre等)，熟悉NFS、CIFS协议者优先；
  4. 熟悉用户态文件系统，有RDMA、SPDK、DPDK开发经验者优先；
  5. 熟悉存储新硬件(AEP，NVMe，NVDIMM)者优先；
  6. 有良好的团队协作能力、学习能力，善于发现和解决问题，对技术有激情；

蚂蚁集团-OceanBase SQL引擎研发工程师/专家-北京/上海/杭州 3年经验

  1. 熟练掌握C/C++语言，良好的编程习惯、数据结构、算法等基础知识
  2. 熟悉关系数据库和/或分布式系统的运行机制和架构体系，熟悉主流的数据库和/或分布式系统的工作原理
  3. 加分项：有SQL执行引擎、优化器、分布式执行引擎等相关项目经验者优先
  4. 加分项：有大型互联网公司底层系统研发经验或者关系数据库研发经验者优先
  5. 良好的团队协作能力，有技术激情，能够胜任有挑战的工作；

云智能-存储研发专家 - 操作系统-杭州/北京 3年经验

  1. 有强烈的技术热情，工作责任感，有较强的对内外沟通能力，具备迅速掌握解决问题所需技术的方法和能力，具备开源社区工作经验者优先；
  2. 具备扎实的计算机专业基础，包括计算机体系结构、操作系统内核、文件系统，存储技术，虚拟化和容器技术等；
  3. 至少可以熟练掌握C/Go/Rust中的一种作为开发语言，并可以在需要时使用其他语言；
  4. 有 Kata Containers, Rust VMM, Linux Kernel, 文件系统，分布式存储，Containerd, Kubernetes, Kraken, Dragonfly，Nydus 等项目上游参与经验者优先，请提供个人的 commits 或 PR 链接；
  5. 有性能观测和性能优化，系统/应用热点探测等方面经验者优先，熟悉perf, ebpf者优先；
  6. 具有大数据、数据分析、挖掘、AIOps相关经验或能力者优先。

云智能-Linux操作系统解决方案架构师-杭州/北京/深圳 5年

  1. 5年以上Linux操作系统研发或架构岗位经历；对Linux操作系统发行版有深入了解；
  2. 具有丰富的项目需求分析、方案架构设计经验；
  3. 具备优秀的技术结构化思维能力，擅长从用户使用场景及产品体验角度出发思考问题，能够从复杂场景中提炼客户需求本质和产品痛点；
  4. 有优秀的文档能力，使用文字、图示清晰的表达架构意图，能够熟练编写各类技术文档；
  5. 实数领域涉猎广泛，知识面广，并在操作系统领域具有专长优势；
  6. 良好的逻辑思维、沟通表达能力，能够跨团队协同完成既定目标；
  7. 有操作系统发行版厂商工作经历者优先。

云智能-分布式系统开发专家-机器学习平台 3年

  1. 有三年以上的C++经验，具有良好的编程习惯，熟悉多线程编程，内存管理，设计模式和Linux/Unix开发环境；
  2. 2年以上分布式系统相关工作经历，复杂系统软件的设计能力和调试能力；
  3. 精通各种大数据计算框架，熟悉Hadoop/HIVE/MPI/Spark/TensorFlow等分布式计算框架；
  4. 较强的团队合作经验，能够很好的团队合作及沟通并达成共赢；
  5. 在机器学习、深度学习、大规模分布式机器学习以及在搜索、广告、推荐、机器翻译等领域有经验者优先；
  6. 熟悉GPU硬件架构，精通CUDA，精通cuDNN，在深度学习计算框架等领域有丰富优化经验者优先；

云智能-分布式研发专家-Java/C++ 3年

  1. 具备扎实的计算机理论基础, 对数据结构及算法有较强的功底
  2. 精通Golang、C++、Java任一语言编程，具备优秀的系统Debug/Profiling能力和经验
  3. 熟悉Yarn/Mesos/Hadoop/HBase/Flink/Spark/Storm/docker等开源技术
  4. 熟悉 Kubernetes/Docker 生态，熟练掌握Kubernetes/容器调度相关技术和相关项目代码实现
  5. 具备实际的大数据业务开发经验，具备大规模集群管理经验者优先
  6. 在以上相关领域有开源社区经验者优先

云智能-飞天云存储产品资深架构师/资深技术专家-北京/杭州 8年

阿里云智能基础产品事业部负责基础产品如弹性计算、存储、网络、安全、容器、中间件等产品的研发。

存储团队负责研发盘古统一存储平台，支撑云存储，数据库，大计算，搜索等产品和服务。同时存储团队也负责研发块存储，对象存储、文件存储和表格存储等多种云产品和服务。

  1. 资深架构师，设计大规模分布式系统，保障系统的高可靠和高可用；或者

  2. 资深技术leader，带领团队，设计和开发高性能的存储索引引擎；或者

  3. 资深技术专家，在性能评测，性能调优，全栈链路（包括网络、协议、文件系统等）等方面提升系统性能和稳定性。

  4. 有计算机或者相关领域的教育背景，有至少8年以上的工作经验；

  5. 有一定的分布式存储，或者大数据计算，或者分布式数据库等领域的研究和工作经历；

  6. 扎实的计算机体系结构和操作系统方面的基础，有丰富的代码开发经验；

  7. 注重技术细节，关注业界和学术界的技术发展趋势。
