> 查缺补漏01

#### 拷贝构造和赋值操作

用其他对象定义新对象都会调用拷贝构造
```cpp
A a(b); // 调用拷贝构造
A a = b;    // 调用拷贝构造
```

赋值
```cpp
A a;
a = b;  // 调用拷贝赋值
```

举例
```cpp
class MyClass
{
public:
    MyClass(int i = 0)
    {
        cout << i;
    }
    MyClass(const MyClass &x)
    {
        cout << 2;
    }
    MyClass &operator=(const MyClass &x)
    {
        cout << 3;
        return *this;
    }
    ~MyClass()
    {
        cout << 4;
    }
};
int main()
{
    MyClass obj1(1), obj2(2);
    MyClass obj3 = obj1;    // 拷贝构造
    obj3 = obj2;    // 调用赋值操作符
    return 0;
}

输出
1223444
```
