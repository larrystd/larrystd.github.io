STL源码:算法

larrystd
2021-07-10
cpp advanced
cpp, stl

概述

STL算法部分主要由头文件<algorithm>,<numeric>,<functional>组成。要使用 STL中的算法函数必须包含头文件 <algorithm>，对于数值算法须包含 <numeric>，<functional> 中则定义了一些模板类，用来声明函数对象。

算法

查找算法 #include<alogorithm>

  • ForwardIt adjacent_find( ForwardIt first, ForwardIt last ): 在iterator对标识元素范围内，查找一对相邻重复元素，找到则返回指向这对元素的第一个元素的ForwardIterator。否则返回last。重载版本使用输入的二元操作符代替相等的判断。

  • count( InputIt first, InputIt last, const T &value ): 利用等于操作符，把标志范围内的元素与输入值比较，返回相等元素个数。

  • count_if( InputIt first, InputIt last, UnaryPredicate p ): 利用输入的操作符，对标志范围内的元素进行操作，返回结果为true的个数。

  • bool binary_search( ForwardIt first, ForwardIt last, const T& value ): 在有序序列中查找value，找到返回true。重载的版本实用指定的比较函数对象或函数指针来判断相等。

  • std::pair<ForwardIt,ForwardIt> equal_range( ForwardIt first, ForwardIt last,const T& value );: 功能类似equal，返回一对iterator，第一个表示lower_bound，第二个表示upper_bound。
    find: 利用底层元素的等于操作符，对指定范围内的元素与输入值进行比较。当匹配时，结束搜索，返回指向该元素的Iterator。

  • InputIt find( InputIt first, InputIt last, const T& value ) 返回找到的第一个元素的迭代器。

  • InputIt find_if( InputIt first, InputIt last,UnaryPredicate p): 使用输入的函数代替等于操作符执行find。

  • ForwardIterator1 search (ForwardIterator1 first1, ForwardIterator1 last1,ForwardIterator2 first2, ForwardIterator2 last2,BinaryPredicate pred);: 给出两个范围，返回一个ForwardIterator，查找成功指向第一个范围内第一次出现子序列(第二个范围)的位置，查找失败指向last1。重载版本使用自定义的比较操作。

  • ForwardIterator search_n ( ForwardIterator first, ForwardIterator last,Size count, const T& val, BinaryPredicate pred );: 在指定范围内查找val出现n次的子序列。重载版本使用自定义的比较操作。

  • 调用lower_bound, upper_bound二分查找, 返回所在位置的迭代器。对于从小到大的序列, lower_bound返回数组中第一个大于等于该元素的下标, upper_bound返回数组中第一个大于该元素的下标。对于vector, 迭代器作差可以得到偏移。

排序和通用算法

排序算法共14个，包含在<algorithm>头文件中，用来判断容器中是否包含某个值，这里只列出一部分算法：

  • OutputIterator merge (InputIterator1 first1, InputIterator1 last1,InputIterator2 first2, InputIterator2 last2,OutputIterator result, Compare comp): Combines the elements in the sorted ranges [first1,last1) and [first2,last2), into a new range beginning at result with all its elements sorted.合并两个有序序列，存放到另一个序列且有序。重载版本使用自定义的比较。

  • void random_shuffle( RandomIt first, RandomIt last );: 对指定范围内的元素随机调整次序。重载版本输入一个随机数产生操作。

  • void nth_element( RandomIt first, RandomIt nth, RandomIt last ): 将范围内的序列重新排序，使所有小于nth的元素的元素都出现在它前面，而大于它的都出现在后面。重载版本使用自定义的比较操作。

  • void reverse (BidirectionalIterator first, BidirectionalIterator last);: 将指定范围内元素重新反序排序。

  • void sort (RandomAccessIterator first, RandomAccessIterator last, Compare comp): 以升序重新排列指定范围内的元素。重载版本使用自定义的比较操作。

The elements are compared using operator< for the first version, and comp for the second. Binary function that accepts two elements in the range as arguments, and returns a value convertible to bool. The value returned indicates whether the element passed as first argument is considered to go before the second in the specific strict weak ordering it defines.
The function shall not modify any of its arguments.
This can either be a function pointer or a function object.

  • void stable_sort ( RandomAccessIterator first, RandomAccessIterator last,Compare comp );: 与sort类似，不过保留相等元素之间的顺序关系，稳定排序。

删除和替换算法 #include <numeric>

删除和替换算法共15个，包含在<numeric>头文件中：

  • OutputIt copy( InputIt first, InputIt last, OutputIt d_first );: 复制序列。

  • ForwardIt remove( ForwardIt first, ForwardIt last, const T& value );: remove 移动了区间中的元素。其结果是，“需要被删除”的元素被移到了区间的尾部。它返回一个迭代器，指向第一个“需要被删除”的元素。在内部，remove 遍历整个区间，用需要保留的元素的值覆盖掉那些要被删除的元素的值。
    唯一可以从容器中删除元素的方法是调用容器的成员函数 erase。remove 算法并不知道它操作的元素的所在容器，所以不可能从容器中删除元素。

  • void replace( ForwardIt first, ForwardIt last,const T& old_value, const T& new_value );将指定范围内所有等于vold的元素都用vnew代替。

  • void swap (T& a, T& b);: 交换存储在两个对象中的值。

  • ForwardIt2 swap_ranges( ForwardIt1 first1, ForwardIt1 last1,ForwardIt2 first2 );: 将指定范围内的元素与另一个序列元素值进行交换。

  • ForwardIt unique( ForwardIt first, ForwardIt last );: 清除序列中重复元素，和remove类似，它也不能真正删除元素。重载版本使用自定义比较操作。

  • OutputIt unique_copy( InputIt first, InputIt last,OutputIt d_first );: 与unique类似，不过把结果输出到另一个容器。

排列组合

  • bool next_permutation (BidirectionalIterator first,BidirectionalIterator last): 取出当前范围内的排列，并重新排序为下一个字典序排列。重载版本使用自定义的比较操作。

  • bool prev_permutation (BidirectionalIterator first,BidirectionalIterator last ): 取出指定范围内的序列并将它重新排序为上一个字典序排列。如果不存在上一个序列则返回false。重载版本使用自定义的比较操作。

数值算法 include <numeric>

  • constexpr T accumulate( InputIt first, InputIt last, T init,BinaryOperation op );: iterator对标识的序列段元素之和，加到一个由val指定的初始值上。重载版本不再做加法，而是传进来的二元操作符被应用到元素上。

  • OutputIt partial_sum( InputIt first, InputIt last, OutputIt d_first ): 创建一个新序列，其中每个元素值代表指定范围内该位置前所有元素之和。重载版本使用自定义操作代替加法。

  • T inner_product( InputIt1 first1, InputIt1 last1,InputIt2 first2, T init );: 对两个序列做内积(对应元素相乘，再求和)并将内积加到一个输入的初始值上。重载版本使用用户定义的操作。

  • OutputIt adjacent_difference( InputIt first, InputIt last,OutputIt d_first );: 创建一个新序列，新序列中每个新值代表当前元素与上一个元素的差。重载版本用指定二元操作计算相邻元素的差。

生成和异变算法

  • void fill( ForwardIt first, ForwardIt last, const T& value ): 将输入值赋给标志范围内的所有元素。

  • void fill_n( OutputIt first, Size count, const T& value ): 将输入值赋给first到first+n范围内的所有元素。

  • UnaryFunction for_each( InputIt first, InputIt last, UnaryFunction f );: 用指定函数依次对指定范围内所有元素进行迭代访问，返回所指定的函数类型。该函数不得修改序列中的元素。

  • void generate( ForwardIt first, ForwardIt last, Generator g );: 连续调用输入的函数来填充指定的范围。

  • OutputIt generate_n( OutputIt first, Size count, Generator g );: 与generate函数类似，填充从指定iterator开始的n个元素。
    generator function object that will be called.

  • OutputIt transform( InputIt first1, InputIt last1, OutputIt d_first,UnaryOperation unary_op );: 将输入的操作作用与指定范围内的每个元素，并产生一个新的序列。重载版本将操作作用在一对元素上，另外一个元素来自输入的另外一个序列。结果输出到指定容器。

关系算法

  • bool equal( InputIt1 first1, InputIt1 last1,InputIt2 first2, InputIt2 last2,BinaryPredicate p );: 如果两个序列在标志范围内元素都相等，返回true。重载版本使用输入的操作符代替默认的等于操作符。

  • bool includes( InputIt1 first1, InputIt1 last1,InputIt2 first2, InputIt2 last2 );: 判断第一个指定范围内的所有元素是否都被第二个范围包含，使用底层元素的<操作符，成功返回true。重载版本使用用户输入的函数。

  • const T& max( const T& a, const T& b );: 返回两个元素中较大一个。重载版本使用自定义比较操作。

  • const T& min( const T& a, const T& b );: 返回两个元素中较小一个。重载版本使用自定义比较操作。

  • ForwardIt max_element( ForwardIt first, ForwardIt last );: 返回一个ForwardIterator，指出序列中最大的元素。重载版本使用自定义比较操作。

  • ForwardIt min_element( ForwardIt first, ForwardIt last );: 返回一个ForwardIterator，指出序列中最小的元素。重载版本使用自定义比较操作。

集合算法

  • OutputIt set_union( InputIt1 first1, InputIt1 last1,InputIt2 first2, InputIt2 last2,OutputIt d_first );: 构造一个有序序列，包含两个序列中所有的不重复元素。重载版本使用自定义的比较操作。

  • set_intersection: 构造一个有序序列，其中元素在两个序列中都存在。重载版本使用自定义的比较操作。

  • set_difference: 构造一个有序序列，该序列仅保留第一个序列中存在的而第二个中不存在的元素。重载版本使用自定义的比较操作。

  • set_symmetric_difference: 构造一个有序序列，该序列取两个序列的对称差集(并集-交集)。

堆算法

  • make_heap: 把指定范围内的元素生成一个堆。重载版本使用自定义比较操作。

  • pop_heap: 并不真正把最大元素从堆中弹出，而是重新排序堆。它把first和last-1交换，然后重新生成一个堆。可使用容器的back来访问被”弹出”的元素或者使用pop_back进行真正的删除。重载版本使用自定义的比较操作。

  • push_heap: 假设first到last-1是一个有效堆，要被加入到堆的元素存放在位置last-1，重新生成堆。在指向该函数前，必须先把元素插入容器后。重载版本使用指定的比较操作。
