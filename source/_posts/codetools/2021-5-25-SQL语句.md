### 命令
#### 持久化
* 导出整个数据库   
`mysqldump -u 用户名 -p --default-character-set=latin1 数据库名 > 导出的文件名(数据库默认编码是latin1)`
`mysqldump -u wcnc -p smgp_apps_wcnc > wcnc.sql`
* 导出表   
`mysqldump -u 用户名 -p 数据库名 表名> 导出的文件名`
* 导入数据库,常用source 命令`source wcnc_db.sql`
  
获取表信息

`create database <数据库名>`    
`show databases`    
`drop database <数据库名>`    
`use <数据库名>`   
`select database();` 查看当前的数据库   
`show tables; ` 查看包含的表

#### 表操作

1. 创建表`create table <表名> ( <字段名1> <类型1> [,..<字段名n> <类型n>]);`

```
mysql> create table MyClass(
> id int(4) not null primary key auto_increment,
> name char(20) not null,
> sex int(4) not null default '0',
> degree double(16,2));

CREATE TABLE IF NOT EXISTS `runoob_tbl`(
   `runoob_id` INT UNSIGNED AUTO_INCREMENT,
   `runoob_title` VARCHAR(100) NOT NULL,
   `runoob_author` VARCHAR(40) NOT NULL,
   `submission_date` DATE,
   PRIMARY KEY ( `runoob_id` )
)ENGINE=InnoDB DEFAULT CHARSET=utf8;
```

获取表结构，`desc 表名`，或者show columns from 表名

2. 删除表，`drop table <表名>`


3. 插入数据
`insert into <表名> [( <字段名1>[,..<字段名n > ])] values ( 值1 )[, ( 值n )]`
```
insert into MyClass values(1,'Tom',96.45),(2,'Joan',82.99), (2,'Wang', 96.59);

INSERT INTO account (name,balance) values('dog',11),('cat',2);
```

4. 查询前几行数据
`select * from MyClass order by id limit 0,2;`

5. 删除表中数据
`delete from 表名 where 表达式`

6. 修改表中数据
`修改表中数据：update 表名 set 字段=新值,… where 条件`

```
update MyClass set name='Mary' where id=1;
```
7. LIKE 子句 ，没有使用百分号 %, LIKE 子句与等号 = 的效果是一样的。
`<>`表示不等于
`IS NULL` 判空
```
SELECT * from runoob_tbl  WHERE runoob_author LIKE '%COM';
```

8. UNION, UNION 操作符用于连接两个以上的 SELECT 语句的结果组合到一个结果集合中。多个 SELECT 语句会删除重复的数据
两个执行UNION的数据，需要保证字段一样。

9. 排序。SELECT 语句使用 ORDER BY 子句将查询数据排序后再返回数据：
`SELECT * from runoob_tbl ORDER BY submission_date ASC;`

连接

可以在 SELECT, UPDATE 和 DELETE 语句中使用 Mysql 的 JOIN 来联合多表查询。连接就是根据某个字段的相同数据将两张表连接起来。
`mysql> SELECT a.runoob_id, a.runoob_author, b.runoob_count FROM runoob_tbl a INNER JOIN tcount_tbl b ON a.runoob_author = b.runoob_author;`

在表中增加字段
`alter table 表名 add 字段 类型 其他;`

```
alter table MyClass add passtest int(4) default '0'
```

更改表名：
`alter table white_user rename to white_user_new ;`

更新字段内容
`alter table white_user change column name nick_name  varchar(50) null comment '昵称';`

```
修改字段类型, 使用modify
ALTER  TABLE 表名 MODIFY [COLUMN] 字段名 新数据类型 新类型长度  新默认值  新注释;

修改字段名，使用change

ALTER  TABLE 表名 CHANGE [column] 旧字段名 新字段名 新数据类型;	

修改表名, 使用rename

ALTER TABLE 旧表名 RENAME TO 新表名 ;

增加字段 使用add
ALTER TABLE 表名 ADD [COLUMN] 字段名 字段类型 是否可为空 COMMENT '注释' AFTER 指定某字段 ;

删除字段 drop
ALTER TABLE 表名 DROP [COLUMN] 字段名 ;
```


类型
```
1．INT[(M)] 型： 正常大小整数类型
2．DOUBLE[(M,D)] [ZEROFILL] 型： 正常大小(双精密)浮点数字类型
3．DATE 日期类型：支持的范围是1000-01-01到9999-12-31。MySQL以YYYY-MM-DD格式来显示DATE值，但是允许使用字符串或数字把值赋给DATE列
4．CHAR(M) 型：定长字符串类型，当存储时，总是是用空格填满右边到指定的长度
5．BLOB TEXT类型，最大长度为65535(2^16-1)个字符。
6．VARCHAR型：变长字符串类型
```

#### 管理

* 开启与关闭    
使用 service 启动： `service mysql start`
2、使用 mysqld 脚本启动：`/etc/inint.d/mysql start`   
使用 service 启动：service mysql stop
2、使用 mysqld 脚本启动：/etc/inint.d/mysql stop

* 修改密码

首先，修改配置跳过密码校验

```
vim /etc/mysql/mysql.conf.d/mysqld.cnf
添加一行skip-grant-tables，跳过密码校验
```
之后重启，修改密码,刷新权限

```
sudo service mysql restart
mysql>use mysql;
mysql>update user set authentication_string=password('新密码') where user='root';
mysql> flush privileges;
mysql> quit;

sudo service mysql restart
```

* 授权

授权命令GRANT 语句的语法如下：
```
GRANT privileges (columns)
ON what
TO user IDENTIFIEDBY "password"
WITH GRANT OPTION

增加一个用户test1密码为abc，让他可以在任何主机上登录，并对所有数据库有查询、插入、修改、删除的权限。
grant select,insert,update,delete on *.* to test1@"%" Identified by "abc";

增加一个用户test2密码为abc, 让他只可以在localhost上登录，并可以对数据库mydb进行查询、插入、修改、删除的操作。
grant select,insert,update,delete on mydb.* to test2@localhost identified by "abc";
```

#### SQL注入
sql注入就是通过输入不合法的sql语句，来获取敏感信息
万能密码，
```
输入
' or 1=1--
' or 1=1 or'
```

`SELECT status FROM Users WHERE username = ' ' or 1=1 or ''` where条件恒成立，相当于`SELECT status FROM Users`

而
`SELECT status FROM Users WHERE username = ' ' or 1=2 union select username from Users where 1=1 or ''`
可以通过`union`执行你想执行的sql语句
