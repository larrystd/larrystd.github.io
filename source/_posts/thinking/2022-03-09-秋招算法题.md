5. leetcode15 三数之和

判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0

通过排序使(a,b,c) 这个顺序会被枚举到，而 (b, a, c)(b,a,c)、(c, b, a)(c,b,a) 等等这些不会，

用双指针优化, 外面一个循环表示第一个元素a, 之后两个指针寻找是否寻找b, c使a+b+c = 0

为了不出现重复三元组, 外层循环要跳过重复数

23. leetcode91. 解码方法

```
一条包含字母 A-Z 的消息通过以下方式进行了编码：

'A' -> 1
'B' -> 2
...
'Z' -> 26
给定一个只包含数字的非空字符串，请计算解码方法的总数。

题目数据保证答案肯定是一个 32 位的整数。
```

动态规划, dp[i]表示的意思是当前字符串的以索引 i 结尾的子问题。

遍历到一个元素的时候，有两种方式来组成编码方式，第一种是这个元素本身（需要自身是[1,9]）,第二种是它和前一个元素组成[10, 26]。dp[i] = 以自身去编码（一位） + 以前面的元素和自身去编码(两位)
