#### threading

创建 Thread 对象,然后让它们运行，每个 Thread 对象代表一个线程

Thread 的构造方法中，最重要的参数是 target，所以我们需要将一个 callable 对象赋值给它，线程才能正常运行。

如果要让一个 Thread 对象启动，调用它的 start() 方法。
```py
import threading
import time

def test():

    for i in range(5):
        print('test ',i)
        time.sleep(1)


thread = threading.Thread(target=test)
thread.start()
```
#### bytes string
python3字符串和二进制数据流做了明确的区分。文本总是Unicode，由str类型表示，二进制数据则由bytes类型表示。Python 3不会以任意隐式的方式混用str和bytes，你不能拼接字符串和字节流，也无法在字节流里搜索字符串（反之亦然），也不能将字符串传入参数为字节流的函数（反之亦然）。   
编码，**编码就是把一个字符用一个二进制来表示**。ASCII码，一种8位即1个字节的编码规范,UTF-8编码应运而生，它规定英文字母系列用1个字节表示，汉字用3个字节表示。   
```py
>>> s = "中文"
>>> s
'中文'
>>> type(s)
<class 'str'>
>>> b = bytes(s, encoding='utf-8')
>>> b
b'\xe4\xb8\xad\xe6\x96\x87'
>>> type(b)
<class 'bytes'>
```
字符串类str里有一个encode()方法，它是从字符串向比特流的编码过程。而bytes类型恰好有个decode()方法，它是从比特流向字符串解码的过程。
```py
>>> b
b'\xe4\xb8\xad\xe6\x96\x87'
>>> type(b)
<class 'bytes'>
>>> s1 = str(b)
>>> s1
"b'\\xe4\\xb8\\xad\\xe6\\x96\\x87'"
>>> type(s1)
<class 'str'>
>>> s1 = str(b, encoding='utf-8')
>>> s1
'中文'
>>> type(s1)
<class 'str'>
```

**必须时刻注意，某个函数处理的是字符串本身还是编码后的二进制**。一般的，数据需要传输或者保存用bytes，数据处理则用str。所以一般的操作是，当数据持久化或网络传输前需要转成bytes。而导出则需要转成字符串。

 #### 特殊方法
 python风格的`__getitem__`和`__len__`特殊方法。
 ```py
 import collections

 Card = collections.namedturple('Card', ['rank', 'suit'])

 class FrenchDeck:
    ranks = [str(n) for n in range(2,11)] + list('JQKA')
    suits = 'spades diamonds clubs hearts'.split()

    def __init__(self):
        self._cards = [Card(rank, suit) for suit in self.suits for rank in self.ranks]
    
    def __len__(self):
        return len(self._cards)
    
    def __getitem__(self, position):
        return self._cards[position]
 ```

 以上, `namedturple`构建`Card`对象，显然，`len()`方法是`__len__`提供的，索引则是由`__getitem__`方法提供的。

 python这种数据模型好处
 * 不必记住方法，例如.size()等、
 * 方便兼容标准库

 特殊方法是被python解释器调用，因此不应该调用`my_object.__len__()`而是`len(my_object)`。
 特殊方法的调用可能是隐式的，例如`for i in x:`语句背后是`iter(x)`，即`__iter__()`方法。

 ```py
class Vector:
    def __init__(self, x=0, y=0):
         self.x = x
         self.y = y
    
    def __repr__(self):
        return 'Vector(%r, %r)' % (self.x, self.y)
    
    def __add__(self, other):
        x = self.x + other.x
        y = self.y + other.y
        return Vector(x,y)

v1 = Vector(1,2)
v2 = Vector(2,3)
v3 = v1+v2

print(v3)

# 输出
Vector(3, 5)
```

`__repr__`可以输出字符串表示形式，类似于java的tostring(), 如果没有实现，以上输出结果`<__main__.Vector object at 0x7fef95f2b240>`
类似的，`__str__`特殊方法在str()函数或者print()中调用。

`__add__`,`__mul__`特殊方法相当于cpp中的`operator+()`, `operator*()`。

特殊方法一览
* 字符串/序列表示，`__repr__`, `__str__`, `__format__`, `__bytes__`
* 数值转换 `__abs__`, `__bool__`, `__int__`, `__float__`, `__hash__`
* 集合模拟 `__len__`, `__getitem__`, `__setitem__`, `__contains__`
* 迭代枚举 `__iter__`, `__reversed__`, `__next__`
* 调用模拟 `__call__`
* 上下文管理 `__enter`, `__exit__`
* 实例创建和销毁 `__new__`, `__init__`, `__del__`
* 属性管理  `__getattr__`, `__setattr__`, `__dir__`
* 算术运算符 `__add__`, `__mul__`, `__and__`等
* 比较运算符 `__lt__`, 小于< `__le__` <=, `__eq__` ==, `__ne__` !=
* 位运算 `__invert__` ~, `__lshift__`<<,  `__and__` &, `__or__`, `__xor__` ^

#### 切片

`s[a:b:c]`表示以c为间隔取值，c为负表示反向取值。而事实上，当使用`seq[start:stop:step]`时，python会调用`seq.__getitem__(slice(start, stop, step))`
```py

```
使用`a[i, j]`时，python调用`a.__getitem__((i, j))`

#### 排序

`list.sort`方法会就地排序，不会把原列表复制一份，返回值为None。而`sorted`相反，会新建一个列表作为返回值。

两者有两可选参数
* reverse, 默认为False, **设置为True则元素降序输出**
* key, 这个函数会作用在序列的每一个元素上，返回的结果时是对比关键字，可以用lambda表达式，也可以设置`key = len`这种基于字符串长度的排序

```py
fruits = ['grape', 'raspberry', 'apple', 'banana']
sorted(fruits, key = len, reverse = True)
fruits.sort()
fruits

['raspberry', 'banana', 'grape', 'apple']
['apple', 'banana', 'grape', 'raspberry']
```

#### 数组

如果需要一个只包含数字的列表，则`array.array`比`list`更高效，因为数组背后存储的不是float对象，而是字节表述，这与C语言一样。

python数组创建需要一个类型码，表示底层C语言存放数据类型，例如`b`代表有符号字符`signed char`

```py
from array import array
from random import random

floats = array('d', (random() for i in range(1000)))
floats[-1]

fp = open('floats.bin', 'wb')
floats.tofile(fp)
fp.close()

floats2 = array('d')
fp = open('floats.bin', 'rb')
floats2.fromfile(fp, 1000)

floats2 == floats
```

快速序列化数字类型还可以用 pickle, 几乎与array.tofile一样块, 但可以处理几乎所有的内置数字类型，包括嵌套集合等。

当然序列类型还有Numpy, SciPy, Pandas等，其高效性归功于背后的C和Fortran代码。
