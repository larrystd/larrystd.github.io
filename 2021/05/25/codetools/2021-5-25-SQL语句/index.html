SQL语句和MYSQL命令

2021.05.25 larrystd tools  热度 ℃

本文主要总结了mysql的常用命令，方便使用。挖个坑，以后专门分析研究mysql架构和内部存储原理。

命令

持久化

  • 导出整个数据库
    mysqldump -u 用户名 -p --default-character-set=latin1 数据库名 > 导出的文件名(数据库默认编码是latin1)
    mysqldump -u wcnc -p smgp_apps_wcnc > wcnc.sql
  • 导出表
    mysqldump -u 用户名 -p 数据库名 表名> 导出的文件名
  • 导入数据库,常用source 命令source wcnc_db.sql

获取表信息

create database <数据库名>
show databases
drop database <数据库名>
use <数据库名>
select database(); 查看当前的数据库
show tables; 查看包含的表

表操作

  1. 创建表create table <表名> ( <字段名1> <类型1> [,..<字段名n> <类型n>]);

获取表结构，desc 表名，或者show columns from 表名

  1. 删除表，drop table <表名>

  2. 插入数据
    insert into <表名> [( <字段名1>[,..<字段名n > ])] values ( 值1 )[, ( 值n )]

  3. 查询前几行数据
    select * from MyClass order by id limit 0,2;

  4. 删除表中数据
    delete from 表名 where 表达式

  5. 修改表中数据
    修改表中数据：update 表名 set 字段=新值,… where 条件

  1. LIKE 子句 ，没有使用百分号 %, LIKE 子句与等号 = 的效果是一样的。
    <>表示不等于
    IS NULL 判空

  2. UNION, UNION 操作符用于连接两个以上的 SELECT 语句的结果组合到一个结果集合中。多个 SELECT 语句会删除重复的数据
    两个执行UNION的数据，需要保证字段一样。

  3. 排序。SELECT 语句使用 ORDER BY 子句将查询数据排序后再返回数据：
    SELECT * from runoob_tbl ORDER BY submission_date ASC;

连接

可以在 SELECT, UPDATE 和 DELETE 语句中使用 Mysql 的 JOIN 来联合多表查询。连接就是根据某个字段的相同数据将两张表连接起来。
mysql> SELECT a.runoob_id, a.runoob_author, b.runoob_count FROM runoob_tbl a INNER JOIN tcount_tbl b ON a.runoob_author = b.runoob_author;

在表中增加字段
alter table 表名 add 字段 类型 其他;

更改表名：
alter table white_user rename to white_user_new ;

更新字段内容
alter table white_user change column name nick_name varchar(50) null comment '昵称';
